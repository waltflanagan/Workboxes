# Resources

Cross-area wiki managed by the agent. Unlike `Areas/`, nothing here is
project-scoped — Resources are shared reference that any area can link to.

## Claude: Read this first

Resources are agent-maintained. When a new person, place, or reference
document is encountered across any area, create or update the relevant
entry here. Link *to* Resources from area notes; never move Resources
*into* an area.

On every interaction with a person document, update `## Interactions`
(reverse-chrono, one bullet per contact) and any fields that have
changed (phone, insurance status, etc.).

  anyone the family interacts with professionally or semi-professionally.
