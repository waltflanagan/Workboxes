# People

CRM-style directory. One file per person. Agent creates and updates entries
whenever a new name surfaces in any area — doctor, therapist, teacher, tutor,
coach, specialist, vendor, etc.

## Claude: Read this first

- One file per person: `<firstname-lastname>.md` (kebab-case).
- If last name is unknown, use `<firstname-descriptor>.md`
  (e.g., `dr-kim-pediatrician.md`).
- Search this directory before creating a new entry — avoid duplicates.
- When a person is mentioned in an area note, link here:
  `[[Resources/People/firstname-lastname|Full Name]]`.
- Keep `## Interactions` as the running log; update it on every new contact.
